## 一、安装前准备

#### 1.PHP服务器环境

	若您使用的PHP服务器为Apache，您直接将本网站源代码（或release）放置到网站根目录即可。

   若您使用的PHP服务器为NGINX，请您将本网站源代码（或release）放置到网站根目录后复制nginx.conf的内容到网站伪静态配置文件。
